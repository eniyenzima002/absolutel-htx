# Absolutely HTX Frontend

A front-end only Next.js starter for an open mic venue website.

## Included
- App Router
- Tailwind CSS
- Home page
- Upcoming Shows page
- Artists page
- Submission page
- Contact page
- Psychedelic hero background, wave separator, and lighter body background

## Run locally
```bash
npm install
npm run dev
```

## Deploy
- Push to GitHub
- Import into Netlify or Vercel
- Build command: `npm run build`
- Publish uses the Next.js adapter automatically on supported hosts

## Notes
Forms are front-end only right now. Connect them later to Netlify Forms, Formspree, or your own API.
